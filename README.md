# Чек-лист VK Calls (Web 1:1)

1. Прогон базовых сценариев в режиме инкогнито.

### Запуск звонка

2. Исходящий аудио звонок из диалога успешен.
3. Исходящий видео звонок из диалога успешен.
4. Нажатие **Выйти** во время дозвона завершает попытку.
5. Повторный звонок сразу после завершения предыдущего успешен.
6. Завершение звонка по истечении времени дозвона (таймаут).
7. Создание звонка и вход в него с одного аккаунта, но с разных браузеров - корректная обработка (отключение в одном браузере при подлючении в другом).

### Входящий звонок

8. Входящий звонок отображается
9. Принять входящий звонок - соединение устанавливается успешно.
10. Отклонить входящий звонок - звонок завершается.
11. Не отвечать на входящий звонок - звонок завершится по истечении времени ожидания.
12. Принять входящий звонок во время активного звонка - первый звонок завершается, начинается другой.
13. Отклонить входящий звонок во время активного звонка - первый звонок продолжается, отклоняется второй звонок.
14. Не отвечать на входящий звонок во время активного звонка - первый звонок продолжается, второй звонок отклоняется по истечении времени ожидания.

### Разрешения браузера (микрофон/камера)

15. При первом звонке появляется запрос разрешения микрофона
16. Разрешить микрофон - звук работает
17. Запретить микрофон - звук не работает, алерт "Ваш микрофон отключен"
18. Включить камеру в аудиозвонке - запрос разрешения камеры
19. Разрешить камеру - видео работает
20. Запретить камеру - отображение аватара собеседника
21. Звонок при ранее отключенном микрофоне/камере - микрофон/камера по умолчанию отключены, при попытке включить микрофон/камеру появляется подсказка как включить устройство

### Управление во время звонка

22. Кнопка **"Выключить микрофон"** отключает передачу звука
23. Кнопка **"Включить микрофон"** возвращает звук
24. Кнопка **"Включить камеру"** передает изображение с камеры (выключена по умолчанию при аудиозвонке)
25. Кнопка **"Выключить камеру"** прекращает передачу изображения, UI сменяется на аватар собеседника
26. Кнока **"Выйти"** (красная кнопка с кретиком) завершает звонок
27. Свернуть/развернуть окно звонка - состояние звонка сохраняется
28. Переключиться на другую вкладку при активном звонке - окно звонка сворачивается и в виде виджета преследует на всех вкладках и звонок не обрывается

### Сеть и стабильность

29. Отключить интернет во время дозвона - появление статуса **"Нет сети"/"Подключение..."**.
30. Отключить интернет во время звонка - появление статуса "подключение", прекращение подключения по истечении времени ожидания.
31. Восстановить интернет во время звонка - звонок восстанавливается.
